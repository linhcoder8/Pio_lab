# Skills Library

Skills = các "khả năng mở rộng" mà worker có thể bật/tắt.

## Format skill

Mỗi skill là 1 folder với `SKILL.md`:

```
skills/
└── my_skill/
    ├── SKILL.md    ← Spec + prompt template
    ├── scripts/    ← Helper scripts (optional)
    └── tools.py    ← Python tool implementations (optional)
```

## Skills được khuyến nghị (Phase 1)

- ⚙️ Code Interpreter
- 🌐 Web Search
- 📁 RAG / Knowledge Base
- 🎨 Image Generation
- 🌐 Browser Automation
- 📅 Long-term Memory

## Tham khảo

Xem các skill có sẵn của Anthropic tại đường dẫn skills của Cowork.
