Skip: chưa làm Engine có UI cho user CRUD rules động. Phase 1 dùng default rules cứng trong file security_policy.yaml: file system (chỉ project folder), API keys (.env), crypto wallet (cấm), Docker (no --privileged). Đủ an toàn cho 1 user. Phase 2-3 upgrade lên Engine khi mở rộng cho team.
